Продемонстировать умение работать с массивом данных(Todo list), полученных с помощью REST API в приложении на Vue.js
Для выполнения требуются: Vue.js, Axios (допускается установка через CDN или однофайловый компонент .vue + npm + Webpack)
Ссылка на REST API (jsonplaceholder): https://jsonplaceholder.typicode.com 
В приложении использовать url: https://jsonplaceholder.typicode.com/todos
Набор стилей CSS - минимальный, адаптивный, желательно использовать Flexbox и/или CSS Grid
Расположение элементов на странице - произвольное

Нужно реализовать:
1.Получение данных (JSON) из REST API и вывод их всех на страницу в виде списка
2.Фильтрация данных полученных из API(данные выводятся согласно выбранных опций):
 - Фильтр по статусу: select с 3 опциями (All, Completed, Uncompleted)
 - Фильтр по ID юзера: с опциями в виде цифр (userId из списка задач) и опция All Users
3.Поиск по ключу title. input, в который при вводе текста, выолняется поиск задач согласно выбранных фильтров и значения этого input  

Дополнительно, но не обязательно
4.На странице добавить блок "Create todo", в нем: 2 поля input(User ID и Title) и кнопка Add (В рамках задачи после успешного ответа от API - добавляем в массив)
